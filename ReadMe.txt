О проекте:
Время на вычисление количества файлов определенных размеров, зависит от их количества и вычислительной мощности устройства. Поэтому, для диска С:\ , который у большинства пользователей  Windows  содержит файлы ОС, количество файлов не считается (показывает 0  0  0) потому, что это может занять несколько минут. 
	Если вы хотите чтобы для диска  С:\  посчитались все файлы, просто уберите или закоментируйте блок if в CountController.cs
	При подсчете файлов не учитываются файлы, у которых атрибуты скрытый или системный. Тоже самое и с каталогами – показываются только те, которые не скрыты и не системные.
